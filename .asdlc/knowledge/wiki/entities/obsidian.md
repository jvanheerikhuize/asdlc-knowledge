# Obsidian

> A popular local-first, Markdown-based note-taking app built around
> **wiki-links** and a graph view — the modern, digital embodiment of
> [[zettelkasten]] principles.

## Relevance to this cluster
Obsidian's linked-Markdown model is the metaphor the
[[llm-wiki-setup-guide-2026|LLM-wiki setup guide]] uses to explain the
[[llm-wiki-pattern|LLM wiki]]: atomic pages, dense wikilinks, and an emergent
[[knowledge-graph]]. This knowledge base uses the same wiki-link syntax and graph
visualisation, applying Obsidian-style linking to
[[knowledge-management-for-llms]].
