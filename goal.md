Goal: create a persistent file-based knowledge base pattern from karpathy.

Requirements:
readable by humans
scaffolds from a single source of truth, the scaffolder can be triggered via a pipeline/action
file based, no infra needed
closest to current industry standards
lives in a subfolder of a repository called .asdlc/knowledge
has a single point of entry for an agent
solution & agent agnostic
needs an ingestion layer for binaries with an interface
needs a visualisation layer
needs an interface to interact with the knowledge base
fact checking and confidence scoring
